### 如何获取key
https://laogou717.com/index.html

###  项目部署
```bash
git clone https://github.com/ramonvc/freegpt-webui.git

git checkout chimeragpt-version  

pip install -r requirements.txt
```
项目根目录创建名为.env文件，写入：
CHIMERA_API_KEY=<API-Key>  
```bash
python run.py
```
